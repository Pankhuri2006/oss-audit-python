## ▶️ How to Run the Scripts

Step 1: Navigate to scripts folder  
cd scripts  

Step 2: Give permission  
chmod +x script1.sh  

Step 3: Run script  
./script1.sh  

Repeat the same steps for all scripts.

---

## 📜 Script Descriptions

Script 1 — System Identity Report  
Displays system details like OS, kernel, user, uptime, and date.

Script 2 — Package Inspector  
Checks whether Python is installed and displays version information.

Script 3 — Disk Auditor  
Shows directory sizes and file permissions using loops and Linux commands.

Script 4 — Python Package Checker  
Checks if a Python package is installed and installs it if not present.

Script 5 — System Resource Monitor  
Displays CPU usage, memory usage, disk usage, and running Python processes.
